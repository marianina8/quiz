READ ME
=======

Steps to run the solution:
1.  Place the goproject build (located under /bin) in a directory
2.  Open terminal
3.  Change diretory (cd) to location of build.
4.  Once in local directory type: ./goproject

or if you have go installed:
1. Place the source (located under /src) in a directory
2. Open terminal
3. Change directory (cd) to location of source file.
4. Once in local direcotry type: go run nodeprime.go
